# 🎬 Movie Recommendation System

> 基于 **Flask + MySQL** 构建的电影推荐系统，提供用户管理、电影管理、评分、观看历史记录以及个性化电影推荐功能。
> 系统采用 **协同过滤（SVD） + 内容相似度** 的混合推荐模型，为用户生成精准的电影推荐。

---

## ✨ 项目特性

### 👥 用户管理

* 用户注册与登录
* 用户信息维护
* 用户数据管理

### 🎥 电影管理

* 添加电影
* 删除电影
* 修改电影信息
* 查询电影
* 支持电影类型分类

### ⭐ 评分系统

* 用户可对电影进行 **1–5 分评分**

* 记录观看完成状态

### 🤖 个性化推荐

系统提供多种推荐策略：

* **SVD 协同过滤推荐**

* **热门电影推荐**
* **混合推荐模型**

系统仪表盘展示：

* 用户数量
* 电影数量
* 评分数量
* 系统运行数据统计

# 📌 推荐算法说明

本系统采用 **混合推荐模型**：

### 1️⃣ 协同过滤（SVD）

利用 Surprise 库训练用户评分矩阵：

```
User × Movie Rating Matrix
```

通过矩阵分解预测用户潜在评分。

### 2️⃣ 内容推荐

根据电影：

* 类型
* 标签
* 相似度

推荐相似电影。

### 3️⃣ 热门推荐

根据：

* 评分数量
* 平均评分

生成热门榜单。

### 4️⃣ 混合推荐

结合：

* 协同过滤
* 内容推荐
* 热门推荐

生成最终推荐结果。
